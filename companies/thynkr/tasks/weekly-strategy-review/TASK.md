Every Monday, review the state of all active projects and align priorities for the week.

## Checklist

1. Review all open tasks across projects — what's done, what's blocked, what's in progress
2. Check if any P1 tasks are at risk of missing the April 20 pilot deadline
3. Review growth metrics from the Growth Lead's weekly report
4. Identify the top 3 priorities for the week
5. Create or reassign tasks as needed
6. Post a weekly summary comment: what shipped last week, what's planned this week, any blockers

## Escalation

If any P1 task is blocked on external input (school contact, legal review), flag it immediately to the board operator (Suren) — don't wait for the weekly review.
